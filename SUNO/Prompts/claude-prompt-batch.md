# Scary Terry Productions — Pseudo Style Box Library Addendum

Built for the **Pseudo** persona: melodic, catchy hood beats with lush reverse-reverb, a slightly demonic undertone, and big-villain energy carrying astounding degrees of arrogance. Each entry has a title, a real description of how it actually sounds, a ~900-character style box, and its own exclusion list. Concepts are wholly distinct from catalog entries 1–80.

**Total entries in this addendum: 30** (catalog numbers 81–110)

> Note on the reverse delivery: the reversed-vocal section is kept clean and undistorted. Processed / distorted vocal FX are fair game on the intro and as texture — just not on the reverse pass.
